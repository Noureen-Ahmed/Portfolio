# Professional Portfolio

A modern, responsive portfolio built with React and Vite, featuring smooth animations with Framer Motion and beautiful iconography from Lucide React.

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
Run the development server:
```bash
npm run dev
```

### Build
Generate a production-ready build in the `dist/` folder:
```bash
npm run build
```

## üåê Live Demo & Deployment

This project is configured for automatic deployment to **GitHub Pages** using GitHub Actions.

### Setup Instructions for GitHub
1. **Upload to GitHub**:
   - Initialize a git repository locally if you haven't: `git init`
   - Add all files: `git add .`
   - Commit: `git commit -m "Initial commit"`
   - Create a repository on GitHub and add it as a remote: `git remote add origin <your-repo-url>`
   - Push to the `main` branch: `git push -u origin main`

2. **Activate GitHub Pages**:
   - Go to your repository on GitHub.
   - Click on **Settings** > **Pages**.
   - Under **Build and deployment** > **Source**, change the dropdown to **GitHub Actions**.
   - The deployment will trigger automatically on your next push, or you can check the **Actions** tab to see the progress.
